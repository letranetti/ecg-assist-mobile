ECG Assist V5 Mobile

MEJORAS
- Segmentación multicanal: oscuridad, color y contraste local.
- Normalización de iluminación.
- Supresión de rejilla por color y proyecciones.
- ROI automático o manual.
- Seguimiento dinámico del trazado.
- Detección QRS por derivada, energía e umbral robusto MAD.
- Detección P-Q-R-S-T con ventanas fisiológicas.
- Medición robusta por mediana de varios latidos.
- Rechazo de valores atípicos mediante MAD.
- Dispersión y número de mediciones válidas.
- Latido mediano.
- Caché V5 independiente.

INSTALACIÓN
Sube todos los archivos a la raíz de GitHub Pages. Abre la URL con ?v=5, desinstala la versión anterior y vuelve a instalar desde Chrome.

LIMITACIÓN
No puede garantizar digitalización perfecta con cualquier fotografía. Usa selección manual de una sola tira horizontal para máxima fiabilidad.
